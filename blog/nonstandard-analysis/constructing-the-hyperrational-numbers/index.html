For this series, I am going to assume the existence of the natural numbers, integers, and rationals denoted $\mathbb{N}$, $\mathbb{Z}$, and $\mathbb{Q}$ respectively. For their construction, you can see my series on constructing the real numbers. For now, we will just use the following definitions.

\[\begin{align} &\mathbb{N} = \{ 0, 1, 2, 3, 4, \ldots \} \\[10pt] &\mathbb{Z} = \{ \ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \} = (-\mathbb{N}) \cup \mathbb{N} \\[10pt] &\mathbb{Q} = \left \{ \frac{n}{m} \ \Big\vert \ n, m \in \mathbb{Z} \text{ and } m \neq 0 \right \} \end{align}\]

Let $\langle q_n \rangle_{n=0}^{\infty} = \langle q_0, q_1, q_2, \ldots, q_n, \ldots \rangle$ be an infinte series of rational numbers. We define the set of all infinite sequences of rational numbers as

\[\mathbb{Q}^{\mathbb{N}} = \left \{ \langle q_n \rangle_{n=0}^{\infty} \ \Big\vert \ q_n \in \mathbb{Q} \right \}\]

We define operations between sequences element-wise. i.e.

\[c \cdot \langle q_n \rangle_{n=m}^{\infty}= \langle c \cdot q_n \rangle_{n=m}^{\infty}\] \[\langle q_n \rangle_{n=m}^{\infty} + \langle p_n \rangle_{n=m}^{\infty} = \langle q_n + p_n \rangle_{n=m}^{\infty}\] \[\langle q_n \rangle_{n=m}^{\infty} \cdot \langle p_n \rangle_{n=m}^{\infty} = \langle q_n \cdot p_n \rangle_{n=m}^{\infty}\]

And you can check that all of these are well-defined. In fact, $(\mathbb{Q}^{\mathbb{N}}, +, \cdot)$ forms a ring. However, it does not form a field since there is no well-defined identity.


Now, we want to define a notion of equivalence between two sequences. We say that $\langle q_n \rangle_{n=0}^{\infty} \simeq \langle p_n \rangle_{n=0}^{\infty}$ if they differ in only finintely many places. Another way to think about it is that eventually these sequences are exactly the same, i.e. $\exists m \ \ \text{s.t.} \ \ \langle q_n \rangle_{n=m}^{\infty} = \langle p_n \rangle_{n=m}^{\infty}$. This notation of equivalence can be made mathematically rigorous via the notation of a filter and an ultrafilter. But I don’t think the formalism really adds anything to the intuitive understanding. Using this idea of equivalence, we can define the equivalence class of sequence $\langle q_n \rangle_{n=0}^{\infty}$ under the $\simeq$ operation.

\[[\langle q_n \rangle_{n=0}^{\infty}] = \{ \langle p_n \rangle_{n=0}^{\infty} \ \Big\vert \ \langle q_n \rangle_{n=0}^{\infty} \simeq \langle p_n \rangle_{n=0}^{\infty} \}\]

the hyperrational numbers as the set of equivalence classes on $\mathbb{Q}^{\mathbb{N}}$ induced by the $\simeq$ operation.

\[{^* \mathbb{Q}} = (\mathbb{Q}^{\mathbb{N}} / \simeq)\]

Therefore, all hyperrational numbers are of the form

\[{u} = [\langle q_n \rangle_{n=0}^{\infty}] \in {^* \mathbb{Q}}\]

Also, we will use the notation

\[{^* q} = [\langle q \rangle_{n=0}^{\infty}]\]

Again, I’m going to skim over some of the formalism. But you can check that all operations (addition, multiplication, etc) are well-defined under this equivalence class. Thus, they are well-defined in the hyperreal numbers. Note that exponentiation is a bit more complicated, so it is given its own section. TODO Link section when it exists
